# mason-registry

> [!CAUTION]
> This is Unofficial!!!

Official registry is [here](https://github.com/mason-org/mason-registry)

## Install

Add `registries` field in setup opts

```lua
require("mason").setup({
    registries = {
        "github:mimikun/mason-registry",
        "github:mason-org/mason-registry",
    },
})
```

> [!IMPORTANT]
> NEED run `:MasonUpdate`
