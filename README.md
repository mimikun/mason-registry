# mason-registry

> [!CAUTION]
> This is Unofficial!!!

## Install

Add `registries` field in setup opts

```lua
require("mason").setup({
    registries = {
        "github:mimikun/mason-registry",
        "github:mason-org/mason-registry",
    },
})
```

> [!IMPORTANT]
> NEED run `:MasonUpdate`
